# Walden Three!
My experience with psychology software has been universally terrible. This will eventually become an app easily deployable to a site like fly.io. It will cover billing, scheduling, treatment plan creation, and data collection and analysis for a team of therapists. Walden 3 is modeled off an ABA workflow which allows for an easy transition to workflows for other fields like one for psychotherapy. 
## Roadmap
### At the moment, we support:
Login and user based authentication.
Treatment plan and goal creation.
Scheduling appointments.

### Current Goals:
Viewing scheduled appointments, check ins, and collecting session data.
Session note review.
Editing goals, schedules, and treatment plans.
Unit testing with axum-test.
End-to-End testing.

### Long Term Goals:
Online graphing tool.
FBA data collection and analysis.
Billing.
